Description

FizzBuzz is a JavaScript Application using loops and conditionals.

Write a Program that prints the numbers 1 to 100.

But for multiples of three print 'Fizz' instead of the number and for multiples of five print 'Buzz'.

For multiples of both "three" and "five" print "FizzBuzz".
